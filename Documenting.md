1. install laravel 12
2. install inertia via composer | composer require inertiajs/inertia-laravel
3. create inertia middleware | php artisan inertia:middleware | register the middleware into app.php
4. install vue3 with inertia | npm install @inertiajs/vue3
5. install vue plugin | npm install -D @vitejs/plugin-vue | add vue dependencies under vite js confige file
